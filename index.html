Nuestros proyectos

Chotuve Android

Via de acceso al usuario mediante la aplicacion de Android. Desarrollada con React Native y Expo
GitHub - Manual de Uusario

Chotuve App Server

Servidor central encargado de la comunicacion con la aplicacion. Desarrollado con Flask y PostgreSQL
GitHub - OpenAPI

Chotuve Auth Server

Servidor encargado de la autenticacion de usuarios con el apoyo Firebase. Desarrollado con Flask y PostgreSQL
GitHub - OpenAPI

Chotuve Media Server

Servidor encargado del almacenamiento y administracion de los recursos Media. Desarrollado con NodeJS, Express y MongoDB
GitHub - OpenAPI

Chotuve Web Front

Front end administrativo, visualizacion de estadisticas y administracion de recursos. Desarrollado con React
GitHub - Acceso Web

Chotuve Landing Page

Esta pagina web. Desarrollada con HTML sin frameworks. Pagina estatica servida por cortesia de GitHub
